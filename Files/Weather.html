WEATHER-INVARIANT OBJECT DETECTION: ENHANCING YOLOV8 WITH ENVIRONMENT ADAPTIVE PREPROCESSING FOR ROBUST PERFORMANCE ACROSS DIVERSE CONDITIONS

Project Overview

Developed an adaptive object detection pipeline for challenging weather conditions by integrating CNN-based weather classification, image enhancement, and YOLOv8. The goal was to improve detection reliability under adverse weather (e.g., fog, rain, snow) by dynamically adjusting preprocessing techniques before running inference

Computer Vision YOLOv8 Roboflow Python OpenCV PyTorch

Technical Implementation

Custom Dataset Creation:

  • Used Roboflow to compile and annotate a dataset of road scenes categorized by weather: rainy, snowy, sunny, foggy, and normal

CNN-Based Weather Classification:

  • Built a lightweight CNN to classify the weather condition of each image prior to detection
  • Trained the model on diverse weather scenarios to ensure generalization across real-world conditions

Adaptive Image Preprocessing:

  • Applied weather-specific filters (e.g., dehazing, glare removal, contrast stretching) based on the classifier's output
  • Used OpenCV to implement preprocessing that enhances visibility without distorting scene feature

YOLOv8 for Object Detection:

  • Trained YOLOv8 on preprocessed images to detect road entities (e.g., vehicles, pedestrians, traffic signals)
  • Evaluated its robustness across different weather types with and without enhancement

Results

CNN Weather Classifier:

  • Validation Accuracy: 81.84%
  • High performance on rainy, snowy, and sunny scenes
  • Slightly lower accuracy for foggy and normal scenes due to reduced visual cues

YOLOv8 Object Detection (Post-Enhancement):

  • mAP@0.5: 93.1%
  • mAP@0.5:0.95: 82.3%
  • Precision & Recall: Over 85%
  • Achieved significant accuracy improvements in fog, rain, and snow, validating the effectiveness of the adaptive preprocessing pipeline

This modular system demonstrated that weather-aware preprocessing can meaningfully boost detection reliability in adverse conditions, making it well-suited for autonomous vehicles and safety-critical vision systems
