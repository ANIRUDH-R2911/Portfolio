WE-PARK SMART PARKING SYSTEM

Project Overview

Designed a smart parking system to reduce urban congestion by enabling real-time monitoring, reservation, and navigation to available parking spots. The project integrated cloud computing, mobile interfaces, and virtual prototyping to simulate a scalable, real-world-ready solution for intelligent parking management.

MIT App Inventor Node-Red IBM Cloud Wokwi MQTT

Technical Implementation

Mobile Application Development:

  • Built a user-friendly mobile app using MIT App Inventor that allows users to view available slots, reserve parking, and navigate to selected spots
  • Focused on intuitive UI/UX design and responsive interaction between frontend and backend

Cloud Integration & Data Flow:

  • Connected the app to a real-time data pipeline using IBM Cloud and Node-RED
  • Implemented MQTT messaging for lightweight and efficient communication between components
  • Enabled live updates of parking slot status through simulated sensor inputs

Virtual Simulation:

  • Used Wokwi to simulate microcontroller-based sensor behavior and data flow for parking occupancy
  • Ensured that simulated sensor states were accurately reflected in the app in near real-time

Results

  • The application successfully provided real-time availability updates, slot reservations, and feedback loops within simulated parking scenarios
  • Demonstrated minimal latency between reservation requests and system confirmation, ensuring a smooth user experience
  • Validated the feasibility of deploying the system with physical sensors and GPS modules in real-world setting
  • Highlighted the potential for smart city integration, making urban parking more efficient, accessible, and less congested
